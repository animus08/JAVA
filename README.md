# Java

Small collection of Java example programs organized by topic for learning.

Top-level folders:

How to use
	- `javac "1. Basics\Numbers\Factorial.java"`
	- `java Factorial`

Quick usage
- Compile a single file (from repo root):
	- `javac "1. Basics\Numbers\Factorial.java"`
- Compile all examples in a folder:
	- `javac "5. OOP\Polymorphism\*.java"`
- Run a class (classpath .):
	- `java -cp . MethodOverriding`
- If a class uses a package, run it with its package name:
	- `java -cp . com.example.MainClass`
